Examples

Example 1

PS C:\> Get-AppVolAppStack [-All]

Return all available AppStacks

Example 2

PS C:\> Get-AppVolAppStack -VolumeID 88,19

Return AppStacks with IDs 88 and 19

Example 3

PS C:\> 88,19|Get-AppVolAppStack

Return AppStacks with IDs 88 and 19 thru the pipe

Example 4

PS C:\> Get-AppVolAppStack -Name office

Return all AppStacks where the name contains “office”

Example 5

PS C:\> Get-AppVolAppStack -Name office -Not

Return all AppStacks where the name NOT contains “office”

Example 6

PS C:\> Get-AppVolAppStack -Name office  -Exact

Return all AppStacks where the name is exactly “office”

Example 7

PS C:\> Get-AppVolAppStack -Path "cloudvolumes"

Return AppStacks that has “cloudvolumes” in the datastore path

Example 8

PS C:\> Get-AppVolAppStack -DataStore iSCSI -Exact

Return AppStacks located on datastore iSCSI

Example 9

PS C:\> Get-AppVolAppStack -FileName office

Return AppStacks where vmdk name contains word office

Example 10

PS C:\> Get-AppVolAppStack -CreatedAt "4/28/2015" –ge

Return AppStacks created after or on 4/28/2015

Example 11

PS C:\> Get-AppVolAppStack -CreatedAt "4/28/2015" -gt

Return AppStacks created after but not on 4/28/2015

Example 12

PS C:\> Get-AppVolAppStack -MountedAt $((get-date).AddDays(-30)) -ge -Not

Return AppStacks not mounted in past 30 days

Example 13

PS C:\> Get-AppVolAppStack -TemplateVersion "2.5.1"

Return AppStacks with template version 2.5.1

Example 14

Return AppStacks that have 2 or more assignments
