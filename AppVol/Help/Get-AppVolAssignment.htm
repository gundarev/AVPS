Examples

Example 1

PS C:\> Get-AppVolAssignment [-all]

Return all assignments

Example 2

PS C:\> Get-AppVolAppStack -Name office |Get-AppVolAssignment

Return all assignments for appstacks that has “office” in the name

Example 3

PS C:\> Get-AppVolAssignment -EntityDn "cn=users,dc=domain,dc=com"

Return assignments for users in specific OU

Example 4

PS C:\> Get-AppVolAssignment -EntitySamAccountName denis -Exact

Return assignments for user “denis”

Example 5

PS C:\> Get-AppVolAssignment -EntityDomain CORP

Return assignments for users in domain CORP

Example 6

PS C:\> Get-AppVolAssignment -EntityType:Computer

Return computer assignments

Example 7

PS C:\> Get-AppVolAssignment -EntityType:OrgUnit

Return OU assignments

Example 8

Return appstacks assigned to a user
